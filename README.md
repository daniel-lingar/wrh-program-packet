## Overview

**What Really Happened (WRH)** is a trauma-literacy framework designed for high-barrier, high-pressure environments like reentry programs, veteran services, and recovery support. It replaces shame with sequence and confusion with clarity by teaching participants to see the patterns they are running and how to change their trajectory.

### Core Components

- **The Survival Map**: A five-part framework (The Water, The Alarm, The Survival Moves, The Rescue Boat, The Way Back) for understanding where patterns come from.
- **The Trajectory Engine**: A mechanical model for how direction changes through pattern, fork, interrupt, repetition, evidence, and direction.
